# StreamBox: Video Streaming Platform

StreamBox is a Video Streaming Platform for an organization to stream courses, events and other videos with an user-friendly interface which has various engaging features.

Features of StreamBox:
  1. Login into the site.
  2. Search Videos.
  3. Find videos in categories.
  4. Viewers can like videos and also comment.
  5. Viewers can mark videos to watch later.
  6. Viewcount on each video.
  7. Users can upload videos and manage them.
  8. Light Mode/Dark Mode.
  9. Push notifications when new videos are uploaded.


## Technologies & Frameworks used
- ReactJS
- NodeJS
- MongoDB
- AWS
